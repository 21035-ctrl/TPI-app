เริ่มได้ใน 1 นาที

ไฟล์เดียว ไม่ต้องลง .NET ไม่ต้องตั้งค่าอะไรเพิ่ม

01

แตกไฟล์แล้วเปิด

ดาวน์โหลด ZIP คลิกขวา → Extract All แล้วดับเบิลคลิก TPI.exe

02

ผ่าน SmartScreen

ถ้ามีหน้าจอสีฟ้าขึ้น คลิก More info แล้ว Run anyway (โปรแกรมยังไม่ได้ code-sign)

03

กด Yes ที่ UAC

โปรแกรมขอสิทธิ์ผู้ดูแลระบบเพื่ออ่านเซ็นเซอร์และปรับแต่งระบบ กด Yes แล้วใช้ได้เลย

ก่อนปรับครั้งแรก โปรแกรมสร้างจุดคืนค่า Windowsให้อัตโนมัติ และสำรองค่าเดิมเป็นไฟล์ .reg ทุกครั้ง
คำถามที่พบบ่อย
ปรับแล้วเครื่องมีปัญหา จะแก้ยังไง?+

โปรแกรมสร้าง จุดคืนค่า (Restore Point) ให้อัตโนมัติก่อนปรับครั้งแรก และสำรองค่าเดิมทุกครั้ง เข้าหน้า ประวัติและการย้อนกลับ กดคืนค่าได้ทีละรายการหรือทั้งหมด รายการที่เสี่ยงจะมีป้ายเตือนและถามยืนยันก่อน

ต่างจากโปรแกรมเร่งเครื่องทั่วไปยังไง?+

TPI เน้นยืดอายุเครื่อง — ลดความร้อน ลดการเขียน SSD ลดงานเบื้องหลัง ไม่ใช่รีดความแรงแลกกับอายุฮาร์ดแวร์ ทุกรายการบอกตรง ๆ ว่าตัวไหนยืดอายุ ตัวไหนแลกอายุ และมีหน้าทดสอบให้ดูจากเครื่องคุณเอง

ทำไม Windows เตือนตอนเปิดไฟล์?+

เพราะโปรแกรมยังไม่ได้เซ็นด้วยใบรับรองดิจิทัล (Code Signing) — Windows SmartScreen จะเตือนไฟล์ที่ยังไม่เป็นที่รู้จักไว้ก่อนเป็นปกติ ไม่ได้แปลว่าไฟล์มีอันตราย คลิก More info → Run anyway เพื่อเปิดใช้งาน

ใช้กับโน้ตบุ๊กหรือเครื่องเก่าได้ไหม?+

ได้ และเหมาะกับเครื่องเก่าที่สุด รองรับ Windows 10/11 (64-bit) แรม 4 GB ขึ้นไป รายการที่ต้องการสเปกขั้นต่ำ โปรแกรมจะกันให้เองอัตโนมัติ

ฟรีไหม เก็บข้อมูลอะไรบ้าง?+

ฟรี ไม่มีโฆษณา ไม่ส่งข้อมูลออกนอกเครื่อง ประวัติ โปรไฟล์ และไฟล์สำรอง เก็บอยู่ในเครื่องคุณเท่านั้น พัฒนาเป็นโครงงานส่งแข่งขัน NSC 2026
