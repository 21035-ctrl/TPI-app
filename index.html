การติดตั้ง

เริ่มได้ใน 1 นาที

ไฟล์เดียวจบ ไม่ต้องติดตั้ง .NET ไม่ต้องตั้งค่าอะไรเพิ่ม

01

โหลดแล้วแตกไฟล์

ดาวน์โหลด ZIP แล้วคลิกขวา → Extract All จากนั้นดับเบิลคลิก TPI.exe

02

ผ่านหน้า SmartScreen

ถ้ามีหน้าจอสีฟ้าขึ้น ให้คลิก More info → Run anyway (โปรแกรมยังไม่ได้ code-sign ซึ่งปกติสำหรับโครงงานนักเรียน)

03

กด Yes ที่ UAC

โปรแกรมขอสิทธิ์ผู้ดูแลระบบเพื่ออ่านเซ็นเซอร์และปรับแต่งระบบ — กด Yes แล้วเริ่มใช้ได้เลย

ทุกอย่างโปร่งใส: ก่อนการปรับแต่งครั้งแรก โปรแกรมสร้างจุดคืนค่า Windows ให้อัตโนมัติ และสำรองค่าเดิมเป็นไฟล์ .reg ทุกครั้ง — ตรวจสอบหรือย้อนเองได้เสมอ
FAQ

คำถามที่พบบ่อย

ปลอดภัยแค่ไหน ถ้าปรับแล้วเครื่องมีปัญหา?+

ปลอดภัยหลายชั้น: โปรแกรมสร้าง System Restore Point อัตโนมัติก่อนปรับครั้งแรก สำรองค่าเดิมเป็นไฟล์ .reg ทุกครั้ง และมีหน้า ประวัติและการย้อนกลับ กดคืนค่าได้ทีละรายการหรือทั้งหมด รายการที่เสี่ยงจะมีป้ายเตือนและถามยืนยันก่อนเสมอ

ต่างจากโปรแกรมเร่งเครื่องทั่วไปยังไง?+

เป้าหมายหลักคือยืดอายุเครื่อง — ลดความร้อน ลดการเขียน SSD ลดงานเบื้องหลัง ไม่ใช่รีดความแรงแลกกับอายุฮาร์ดแวร์ ทุกรายการมีป้ายบอกตรง ๆ ว่าตัวไหน "ยืดอายุ" ตัวไหน "แลกอายุ" และมีหน้าวัดผลให้พิสูจน์ด้วยเครื่องของคุณเอง

ทำไม Windows เตือนตอนเปิดไฟล์?+

เพราะโปรแกรมยังไม่ได้ซื้อใบรับรอง code-signing (มีค่าใช้จ่ายสูงสำหรับโครงงานนักเรียน) หน้า SmartScreen จึงขึ้นเตือนตามปกติ — คลิก More info → Run anyway เพื่อใช้งาน

ใช้กับโน้ตบุ๊ก / เครื่องเก่าได้ไหม?+

ได้ และเหมาะมาก — เครื่องเก่าคือกลุ่มที่เห็นผลชัดที่สุด รองรับ Windows 10/11 (64-bit) RAM 4 GB ขึ้นไป รายการที่ต้องการสเปกขั้นต่ำ (เช่น RAM ≥ 8 GB) โปรแกรมจะตรวจและกันให้เองอัตโนมัติ

เสียเงินไหม เก็บข้อมูลอะไรบ้าง?+

ฟรี ไม่มีโฆษณา ไม่ส่งข้อมูลออกนอกเครื่อง — ข้อมูลทั้งหมด (ประวัติการปรับแต่ง โปรไฟล์ ไฟล์สำรอง) เก็บอยู่ในเครื่องของคุณเท่านั้น พัฒนาเป็นโครงงานเข้าแข่งขัน NSC 2026
